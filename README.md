# homepage
Home page for Edward A. Lee

Hosted at https://ptolemy.berkeley.edu/~eal/

## Local Preview

This site uses Server Side Includes (SSI) for headers and footers. To preview locally with SSI support:

1. Run the included server script from this directory:
   ```bash
   python serve.py
   ```

2. Open your browser to http://localhost:8000/~eal/index.html

3. Press Ctrl+C to stop the server.

You can also specify a different port: `python serve.py 3000`
